# 🔐 AI Phishing Website Detection System

## 📌 Description
This project detects phishing websites using Machine Learning and Flask.

## 🚀 Features
- Detect phishing URLs
- ML prediction
- Fake domain detection
- User-friendly UI

## 🛠 Technologies Used
Python, Flask, Scikit-learn, HTML, CSS

## ▶️ How to Run
pip install -r requirements.txt
python app.py
